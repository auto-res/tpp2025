# TPPmark14

**Date:** November 17, 2014

Let `N = {0, 1, 2, 3, ...}` be the set of natural numbers, and let `p, q` range over `N`. We write `(p mod q) = r` to mean that there exist `k` and `r` in `N` such that `p = kq + r` and `0 <= r < q`. We write `(q | p)` when `(p mod q) = 0`.

Prove the following facts:

1. For every `a` in `N`, `(a^2 mod 3)` equals either `0` or `1`.
2. For every `a, b, c` in `N`, if `a^2 + b^2 = 3 c^2`, then `(3 | a)`, `(3 | b)`, and `(3 | c)`.
3. For every `a, b, c` in `N`, if `a^2 + b^2 = 3 c^2`, then `a = b = c = 0`.
