# TPPmark18

有限列 `l = [a0, a1, ..., a_{k-1}]`（長さ `k >= 1`）が **妥当 (valid)** であるとは、次を満たすこととする。

> 任意の自然数 `m, n` について、もし `m + a_m mod k = n + a_n mod k` ならば `m = n`

例として `[1, 1, 1]`, `[2, 0]`, `[1, 5, 3]`, `[2, 0, 1, 9]` は妥当であり、`[1, 2, 1]`, `[3, 0]`, `[1, 3, 5]`, `[2, 0, 1, 8]` は妥当でない。

## 課題

1. 与えられた列が妥当かどうか判定する関数 `is_valid` を定義し、その判定が正しいことを形式的に証明せよ。
2. 妥当な列 `l` に対して、自然数 `n` を用いて `Across(n) = { m | m <= n < m + a_m mod k }` を定義する。十分大きな `n` について、`l` の要素和が `Across(n)` の要素数と `k` の積に等しいことを証明せよ。
