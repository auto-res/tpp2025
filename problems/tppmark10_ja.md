# TPPmark10

**日付:** 2010年10月29日

一次元の盤上で三目並べや五目並べ型のゲームを考え、勝利条件の違いによって攻撃側と防御側の戦略がどのように変化するかを調べる。

## 1. 線形三目並べ

ゲームは整数直線 `Z` 上で行う。攻撃側と防御側の 2 人が交互に、まだ取られていない整数位置を確保する。攻撃側が先手で、`x`, `x + 1`, `x + 2` のように 3 つの連続した整数を確保できれば勝利となる。防御側は攻撃側にそのような連続 3 点を作らせない戦略を持てばよい。

- **課題 1.a:** 防御側が必勝戦略を持つことを証明せよ。

## 2. 等差三目並べ

こちらも整数直線 `Z` 上で行い、2 人が空いている整数を交互に取る。攻撃側が先手で、`x`, `x + d`, `x + 2d`, …, `x + (n - 1)d`（`d > 0`）のように等差数列をなす `n` 個の位置を確保できれば勝利である。防御側は攻撃側にそのような等差数列を作らせないことを目指す。

- **課題 2.a:** `n = 3` および `n = 4` では、攻撃側が常に勝てることを示せ。
- **課題 2.b:** `n = 5` の場合にも攻撃側が勝てることを示せ（これは予想されている）。
- **課題 2.c:** `n > 6` の場合、どちらのプレイヤーが勝利戦略を持つか決定し、根拠を示せ。

**注意:** 等差三目並べでは、整数直線 `Z` を有理数直線 `Q` に置き換えても有限ゲームの結果は変わらない。
