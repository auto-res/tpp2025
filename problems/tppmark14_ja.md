# TPPmark14

**日付:** 2014年11月17日

`N = {0, 1, 2, 3, ...}` を自然数全体の集合とし、`p`, `q` は `N` の元とする。`(p mod q) = r` は「自然数 `k`, `r` が存在して `p = kq + r` かつ `0 <= r < q` が成り立つ」ことを意味する。`(q | p)` は `(p mod q) = 0` を意味する。

次の事実を証明せよ。

1. 任意の `a` ∈ `N` に対して、`(a^2 mod 3)` は `0` または `1` である。
2. 任意の `a, b, c` ∈ `N` に対して、`a^2 + b^2 = 3 c^2` ならば `(3 | a)`, `(3 | b)`, `(3 | c)` が成り立つ。
3. 任意の `a, b, c` ∈ `N` に対して、`a^2 + b^2 = 3 c^2` ならば `a = b = c = 0` である。
